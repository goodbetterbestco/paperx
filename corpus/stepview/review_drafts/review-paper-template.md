# Review Paper Template

This file records the formatting rules for reconstructed paper review drafts.
It is the human-readable source of truth for template updates that we want to
keep stable across corpus review work.

## First Page

The first page should follow these rules:

0. Render on `8.5in x 11in` letter paper.
1. Leave enough vertical space so the paper title starts about `1.6in` from
   the top of the page.
2. Center the paper title and set it in `20pt` regular.
   Add a `*` marker at the end of the title to connect it to the first-page
   support footnote.
3. Render the author line as a centered block with `1in` of horizontal space
   between the author names, at `12pt`.
4. Leave roughly double the earlier vertical spacing between the author line
   and the academic affiliation information.
5. Render the academic department centered at `12pt` with standard line spacing.
6. Render the institution name centered at `12pt` with standard line spacing.
7. Render the institution address centered at `12pt` with standard line spacing.
8. Leave one blank line before the abstract heading.
9. Render `Abstract` centered at `16pt` regular weight.
10. Leave one blank line before the abstract body.
11. Render the abstract paragraph at `11pt`, fully justified, within `1in`
    left and right page margins, with the first line indented about `0.25in`.
    Keep the line spacing slightly looser than the body text and add very
    slight positive letterspacing.
12. Put the funding/support line in the first-page footer as one justified
    paragraph in `9pt` regular.
13. Prefix the footer text with `*`.
14. Place the footer under a horizontal rule that starts at the left border and
    extends about `2.5in`.
15. Put the footer text at `1in` from the page bottom.
16. Put the centered page number at `0.75in` from the page bottom.
17. Keep the main text area on a `1in` bottom margin.

## Section Headings

The body section headings should follow these rules:

1. Use `12pt` bold for section headings.
2. Set the section number flush left.
3. Put the section title about `0.35in` to the right of the section number.
4. Use a New York Times style serif face for section headings and body text.
5. Add centered `11pt` regular page numbers on every page at `0.75in` from the
   page bottom.

## Scope

These rules currently cover only the first page template. Later updates should
extend this file section by section instead of scattering template decisions
across chat history or one-off renderer changes.
